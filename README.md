# library

> A simple book library to store all the books you've read or you're planning to read

## Built with

- HTML
- SASS
- JavaScript

## Features

- Fully responsive
- Form for adding new books
- View books in a table
- Store books in `localStorage`
- Custom context menu to delete the book or change it's read status
